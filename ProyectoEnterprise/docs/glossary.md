# Industrial OT/IT Glossary

| Term | Definition |
| :--- | :--- |
| **SCL** | Structured Control Language. A high-level language for Siemen's PLC programming (similar to Pascal). |
| **OT** | Operational Technology. Hardware and software that detects or causes a change, through the direct monitoring and/or control of industrial equipment. |
| **IT** | Information Technology. |
| **HMI** | Human-Machine Interface. |
| **OPC UA** | Open Platform Communications Unified Architecture. A machine-to-machine communication protocol for industrial automation. |
| **Edge Gateway** | A device or service that acts as the bridge between OT and IT layers. |
| **Stateless** | A design principle where the server does not store any session data about the client. |
