# Industrial Edge Layer

The Bridge between the industrial floor and the Enterprise cloud.

## Functions
- **Protocol Translation**: Converting S7/OPC UA data into JSON/MQTT.
- **Data Buffering**: Ensuring no data loss during network intermittentcy.
- **Initial Validation**: Sanitizing sensor input before it reaches the IT services.

## Components
- `plc-gateway`: A high-performance Python/Go service optimized for low-latency communication with the PLC.
